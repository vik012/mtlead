# MTLead - Marketing Website

A clean, professional static marketing website for mtlead.in.

## 🎯 What's Included

✅ **Hero Section** - Catchy headline & CTA
✅ **4 Key Features** - Simple & clean layout
✅ **Contact Section** - Email + phone links
✅ **Fully Responsive** - Mobile, tablet, desktop
✅ **Smooth Animations** - Professional feel
✅ **Zero bloat** - Just HTML, CSS, JS

## 📝 Files

- **index.html** - Main website
- **styles.css** - All styling
- **script.js** - Smooth scrolling

## ⚙️ Customization

### Update Your Contact Info
Open `index.html` and change:
- `souviksengupta@mtlead.in` → your email
- `(800) 555-1234` → your phone

### Change Company Name
Search & replace "MTLead" with your company name in:
1. `<title>` tag
2. `.nav-brand`
3. Footer

### Customize Colors
Edit the `:root` variables in `styles.css`:
- `--primary` - Main color
- `--accent` - CTA button color

## 🚀 Deploy in 1 Minute

### Netlify (Recommended)
1. Go to https://netlify.com
2. Drag & drop the `website` folder
3. Done!

### Vercel
1. Go to https://vercel.com
2. Import the folder
3. Deploy

### GitHub Pages
Push to GitHub and enable Pages in settings.
